# 金融术语示例

本目录仅展示有限的公开术语示例，帮助读者理解小隐寺投资百科的覆盖范围。它不是完整数据集，也不构成开放数据授权。

| 术语 | 英文 | 简要说明 | 深入阅读 |
| --- | --- | --- | --- |
| 盘前盘后 | Pre-Market / After-Hours | 常规交易时段之外的延长交易，通常流动性更低、价差更大 | [美股入门](https://xiaoyinsi.com/wiki/stocks) |
| 做空 | Short Selling | 借入证券卖出，并计划以后买回归还；亏损理论上可能没有上限 | [美股入门](https://xiaoyinsi.com/wiki/stocks) |
| 期权 | Option | 在约定条件下买入或卖出标的资产的权利，不等同于必须执行的义务 | [期权进阶](https://xiaoyinsi.com/wiki/options) |
| 隐含波动率 | Implied Volatility | 由期权市场价格反推的波动率预期，是定价输入而非方向预测 | [期权进阶](https://xiaoyinsi.com/wiki/options) |
| 永续合约 | Perpetual Futures | 没有固定到期日、通常通过资金费率贴近现货价格的衍生品 | [加密货币](https://xiaoyinsi.com/wiki/crypto) |
