# Protective Put 成本怎么控制？

Protective Put 可以保护股票下跌风险，但保护不是免费的。

控制成本可以从这些方向入手：

1. 选择更低行权价，接受部分下跌。
2. 选择更短保护周期，只覆盖关键事件。
3. 用 Collar 卖出 Call 抵消部分成本。
4. 分批买保护，而不是长期一直满保护。
5. 比较保护成本占组合收益的比例。

保护性 Put 适合在你想继续持有股票，但又不想承担极端下跌时使用。

如果长期不断买保护，成本可能吞掉大部分收益。
