# 资料来源与事实核验规范

## 来源优先级

1. 法律、监管机构和政府文件；
2. 证券交易所、清算机构及官方规则；
3. 上市公司监管披露与项目官方技术文档；
4. 经同行评审的研究和可靠行业数据；
5. 主流媒体及二手解释材料。

能够获得一手资料时，不使用二手文章替代关键事实来源。

## 常用一手来源

- [美国证券交易委员会 SEC](https://www.sec.gov/)
- [美国金融业监管局 FINRA](https://www.finra.org/)
- [美国商品期货交易委员会 CFTC](https://www.cftc.gov/)
- [美国国税局 IRS](https://www.irs.gov/)
- [纽约证券交易所 NYSE](https://www.nyse.com/)
- [Nasdaq](https://www.nasdaq.com/)
- [芝加哥期权交易所 Cboe](https://www.cboe.com/)
- [期权清算公司 OCC](https://www.theocc.com/)

## 核验规则

- 记录来源标题、URL、发布机构和访问日期；
- 对交易时间、结算周期、保证金和监管要求标注适用市场；
- 对可能变化的信息设置复核日期；
- 不把历史表现表达为未来收益承诺；
- 示例数字必须标明是假设值还是实际市场数据；
- 加密项目资料应区分链上事实、项目方声明和第三方观点。
