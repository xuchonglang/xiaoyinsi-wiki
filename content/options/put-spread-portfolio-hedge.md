# Put Spread 如何做组合保护？

Put Spread 用买入 Put 加卖出更低行权价 Put 的方式，降低对冲成本。

它适合用来保护组合的一段下跌，而不是保护无限下跌。

设计时要看：

1. 用哪个指数或 ETF 对冲组合。
2. 到期日覆盖多长时间。
3. 买入 Put 的行权价从哪里开始保护。
4. 卖出 Put 的行权价在哪里停止保护。
5. 对冲比例是否过高或过低。

Put Spread 比单买 Put 便宜，但保护有上限。

新手要把它当保险，不要把对冲仓位变成新的投机主仓。
