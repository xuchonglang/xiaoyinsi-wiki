# FLEX 期权是什么？

FLEX 期权是在交易所和清算框架内，让交易者协商部分合约条款的期权。可定制内容可能包括行权价、到期日、行权方式或结算方式。

它结合了场内清算与一定程度的定制能力，主要用于机构对冲特殊期限或金额。代价是报价透明度和流动性可能不如标准挂牌合约。

接触 FLEX 期权时，应明确：

- 合约乘数、到期和结算条款；
- 报价来源以及能否提前退出；
- 保证金、估值和清算规则；
- 标准期权是否已经能满足需求。

对初学者而言，标准化且流动性好的合约通常更容易控制成交成本。
