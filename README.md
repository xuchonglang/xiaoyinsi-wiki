小隐寺面向中文美股与加密货币投资者，提供结构化的基础知识、交易概念与风险教育。

这个仓库是小隐寺投资百科的官方公开索引，主要用于：

- 整理美股、期权与加密货币领域的核心知识框架；
- 发布适合开发者使用的公开工具与有限示例；
- 接收错别字、失效来源和事实性错误反馈；
- 为读者提供小隐寺官网及投资百科的可信入口。

> 本仓库不提供个性化投资建议，也不承诺任何收益。金融产品存在本金损失风险，使用杠杆和衍生品可能放大亏损。

## 内容原则

1. **先讲风险，再讲用途**：不将高风险产品包装成低风险机会。
2. **区分事实与观点**：规则、数据和观点分别标注。
3. **优先引用原始来源**：监管规则优先引用监管机构或交易所。
4. **说明更新时间**：可能变化的交易规则、费率和政策必须标注核验日期。
5. **提供完整上下文**：不使用脱离条件的收益截图或结论。

详细标准见 [资料来源与核验规范](sources/README.md)。
