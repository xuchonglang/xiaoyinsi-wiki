# 期权限价单 Cancel/Replace 怎么用？

Cancel/Replace 是撤销原订单并提交新价格，但原订单在确认撤销前仍可能成交。

新手最常见错误，是觉得旧单没成交，就直接再下一张更高价的新单，结果两张都成交。

安全调价流程：

1. 使用券商的 Cancel/Replace 功能，而不是重复下新单。
2. 等订单状态明确为取消或替换成功。
3. 部分成交后，只调整剩余数量。
4. 设置最高追价或最低收款边界。
5. 快速行情中先看持仓，不只看订单列表。

Pending Cancel 不代表订单已经失效。

期权调价的目标是改善成交，不是让追价破坏原本的风险回报。
