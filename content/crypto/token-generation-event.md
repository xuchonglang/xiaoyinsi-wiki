# 代币生成事件 TGE 前要检查什么？

TGE（Token Generation Event）通常指代币合约部署、分配、领取或开始流通的阶段。它不等于项目已经成熟，也不保证上线后有足够流动性。

## 用数字代替热度

- 初始流通量占总供应量多少；
- 团队、投资人和社区代币何时解锁；
- 初始市值与完全稀释估值（FDV）；
- 做市和交易池深度；
- 代币是否真的需要，以及需求来自哪里；
- 合约地址与领取入口是否由官方多渠道确认。

低流通、高 FDV 的代币容易在后续解锁中承受卖压。领取时还要防范假网站和恶意授权，先核对域名、链、合约与签名内容。
