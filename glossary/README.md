# 金融术语

本目录整理小隐寺投资百科中适合初学者优先掌握的核心术语。它不是完整数据集，也不构成开放数据授权；完整解释、案例和风险说明请进入对应 Wiki 词条继续阅读。

## 美股与公司分析

| 术语 | 英文 | 简要说明 | 深入阅读 |
| --- | --- | --- | --- |
| 盘前盘后交易 | Pre-Market / After-Hours | 常规交易时段之外的延长交易，通常流动性更低、价差更大 | [盘前盘后交易](../content/stocks/premarket-afterhours.md) |
| 市价单 / 限价单 | Market Order / Limit Order | 市价单优先成交，限价单控制价格；新手应理解滑点和部分成交 | [订单类型](../content/stocks/market-order-limit-order.md) |
| 买卖价差 | Bid-Ask Spread | 买价和卖价之间的差额，是交易成本和流动性的重要体现 | [买卖价差](../content/stocks/bid-ask-spread.md) |
| ETF | Exchange-Traded Fund | 像股票一样交易的一篮子资产工具，可用于分散投资 | [ETF](../content/stocks/etf.md) |
| 市值 | Market Capitalization | 股价乘以总股数，用于衡量公司市场价值规模 | [市值](../content/stocks/market-cap.md) |
| 财报 | Earnings Report | 公司定期披露经营和财务表现，是分析股票的基础材料 | [财报](../content/stocks/earnings-report.md) |
| 10-K 年报 | Form 10-K | 美国上市公司年度报告，包含业务、风险、财务报表和附注 | [10-K 年报](../content/stocks/annual-report-10k.md) |
| EPS | Earnings Per Share | 每股收益，常用于估值，但容易受回购、稀释和一次性项目影响 | [EPS](../content/stocks/eps.md) |
| PE | Price-to-Earnings Ratio | 市盈率，表示市场愿意为每 1 元利润支付多少价格 | [市盈率 PE](../content/stocks/pe-ratio.md) |
| 自由现金流 | Free Cash Flow | 公司经营现金流扣除资本开支后剩余的现金能力 | [自由现金流](../content/stocks/free-cash-flow.md) |
| ROIC | Return on Invested Capital | 衡量公司投入资本创造税后经营利润的能力 | [ROIC](../content/stocks/roic.md) |
| WACC | Weighted Average Cost of Capital | 公司债务和股权资金成本的加权平均，常用于 DCF 折现率 | [WACC](../content/stocks/wacc.md) |
| 股权稀释 | Share Dilution | 公司增发或股权激励导致每股权益被摊薄 | [股权稀释](../content/stocks/share-dilution.md) |
| 止损 | Stop Loss | 预设离场条件以限制亏损，但不能保证跳空时按指定价格成交 | [止损](../content/stocks/stop-loss.md) |
| 保证金账户 | Margin Account | 可借入资金或证券交易的账户，会带来利息、追缴保证金和强平风险 | [保证金账户](../content/stocks/margin-account.md) |

## 期权与衍生品

| 术语 | 英文 | 简要说明 | 深入阅读 |
| --- | --- | --- | --- |
| Call / Put | Call / Put Option | Call 是买入权，Put 是卖出权；期权买方拥有权利，卖方承担义务 | [Call 与 Put](../content/options/calls-and-puts.md) |
| 行权价 | Strike Price | 期权合约约定买入或卖出标的资产的价格 | [行权价](../content/options/strike-price.md) |
| 到期日 | Expiration Date | 期权权利结束的日期，越接近到期时间风险通常越集中 | [到期日](../content/options/expiration-date.md) |
| 权利金 | Option Premium | 期权买方支付、卖方收取的合约价格 | [期权权利金](../content/options/option-premium.md) |
| 实值 / 平值 / 虚值 | ITM / ATM / OTM | 描述期权行权价与标的当前价格的相对关系 | [Moneyness](../content/options/moneyness.md) |
| 隐含波动率 | Implied Volatility | 由期权市场价格反推的波动率预期，是定价输入而非方向预测 | [隐含波动率](../content/options/implied-volatility.md) |
| Delta | Delta | 衡量标的价格变化对期权价格的影响，也可近似理解方向敞口 | [Delta 和 Gamma](../content/options/delta-gamma.md) |
| Gamma | Gamma | 衡量 Delta 对标的价格变化的敏感度，临近到期时可能快速放大 | [Gamma 风险](../content/options/gamma-risk.md) |
| Theta | Theta | 时间流逝对期权价值的影响，买方通常承受时间损耗 | [Theta](../content/options/theta.md) |
| Vega | Vega | 隐含波动率变化对期权价格的影响 | [Vega](../content/options/vega.md) |
| IV Crush | IV Crush | 事件落地后隐含波动率快速下降，可能导致方向看对仍亏钱 | [IV Crush](../content/options/iv-crush.md) |
| 备兑开仓 | Covered Call | 持有正股同时卖出 Call，用权利金换取部分上行空间 | [Covered Call](../content/options/covered-call.md) |
| 保护性 Put | Protective Put | 持有资产同时买入 Put，用期权成本换取下跌保护 | [Protective Put](../content/options/protective-put.md) |
| 垂直价差 | Vertical Spread | 同到期日、不同执行价的两腿期权组合，用于限定收益和风险 | [垂直价差](../content/options/vertical-spread.md) |
| 指派 | Assignment | 期权卖方被要求履行合约义务，可能产生股票和现金交割 | [行权与指派](../content/options/exercise-assignment.md) |

## 加密货币与链上风险

| 术语 | 英文 | 简要说明 | 深入阅读 |
| --- | --- | --- | --- |
| 比特币 | Bitcoin | 最早的大规模去中心化加密资产，核心特征是固定发行规则和 PoW 共识 | [比特币](../content/crypto/bitcoin.md) |
| 以太坊 | Ethereum | 支持智能合约和去中心化应用的公链网络 | [以太坊](../content/crypto/ethereum.md) |
| 稳定币 | Stablecoin | 试图锚定法币或其他资产价格的链上代币，但仍有储备、赎回和脱锚风险 | [稳定币](../content/crypto/stablecoin.md) |
| USDT / USDC / DAI | Stablecoins | 常见美元稳定币，机制、发行方和风险边界不同 | [USDT、USDC、DAI](../content/crypto/usdt-usdc-dai.md) |
| 永续合约 | Perpetual Futures | 没有固定到期日、通常通过资金费率贴近现货价格的衍生品 | [永续合约](../content/crypto/perpetual-futures.md) |
| 资金费率 | Funding Rate | 多空双方周期性支付的费用，用于让永续合约价格贴近现货 | [资金费率](../content/crypto/funding-rate.md) |
| 爆仓 / 强平 | Liquidation | 保证金不足时仓位被系统强制平仓，杠杆越高越容易触发 | [爆仓和强平](../content/crypto/liquidation.md) |
| 热钱包 | Hot Wallet | 联网环境中使用的钱包，操作方便但暴露面更大 | [热钱包](../content/crypto/hot-wallet.md) |
| 冷钱包 | Cold Wallet | 私钥离线保存的钱包方案，适合长期资产但需要备份纪律 | [冷钱包](../content/crypto/cold-wallet.md) |
| 助记词 | Seed Phrase | 恢复钱包控制权的核心秘密，泄露后资产可能被直接转走 | [助记词](../content/crypto/seed-phrase.md) |
| 私钥 | Private Key | 控制地址资产的关键凭证，不应发送给任何人或输入不可信网页 | [公钥和私钥](../content/crypto/public-private-key.md) |
| 钱包授权 | Wallet Approval | 允许合约转移特定资产的权限，过大授权会放大被盗损失 | [钱包授权](../content/crypto/wallet-approval.md) |
| Gas Fee | Gas Fee | 链上交易或合约执行需要支付的网络费用 | [Gas Fee](../content/crypto/gas-fee.md) |
| 智能合约 | Smart Contract | 链上自动执行的程序，存在代码、权限和交互风险 | [智能合约](../content/crypto/smart-contract.md) |
| DEX | Decentralized Exchange | 去中心化交易场所，通常通过流动性池和智能合约完成兑换 | [DEX](../content/crypto/dex.md) |
| 跨链桥 | Cross-Chain Bridge | 在不同链之间转移或映射资产的工具，常见风险包括假币、合约和流动性 | [跨链桥](../content/crypto/cross-chain-bridge.md) |
| DeFi | Decentralized Finance | 链上借贷、交易、收益聚合等金融应用集合，风险来自合约、预言机和清算 | [DeFi](../content/crypto/defi.md) |
| Token 解锁 | Token Unlock | 早期投资人、团队或生态激励代币进入流通，可能影响供给和价格 | [代币解锁](../content/crypto/token-unlock.md) |
