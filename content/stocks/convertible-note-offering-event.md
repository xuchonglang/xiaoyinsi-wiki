# 公司发行可转债后股价为什么会跌？

可转债兼具债务和未来转股权。发行公告可能带来稀释预期，也可能引发套利卖压。

公告后检查：

1. 发行规模和票息。
2. 转换价和转换溢价。
3. 未来最多可能增加多少股份。
4. 是否有 Capped Call 降低部分稀释。
5. 募集资金用途。

低票息不代表没有成本，因为投资者拿到了未来转股权。

如果公司用可转债补充高质量增长资金，影响可能可控；如果用来填现金窟窿，股东要同时承担债务和稀释风险。
