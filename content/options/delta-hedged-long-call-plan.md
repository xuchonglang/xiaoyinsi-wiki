# 买入 Call 后如何做 Delta 对冲？

买入 Call 本质上是看多方向，同时拥有正 Gamma 和正 Vega，但每天会承受 Theta 损耗。

Delta 对冲是用股票或其他工具降低方向敞口，让交易更接近波动率交易。

新手要先写清：

1. 为什么要对冲，而不是简单买 Call。
2. 对冲频率是按时间、价格，还是 Delta 变化。
3. 每次调整的交易成本是多少。
4. 标的流动性是否足够。
5. 到期前什么时候彻底平仓。

对冲不是免费午餐。调仓太频繁会被手续费和滑点吃掉，调仓太少又可能失去保护。

如果你只是想表达看涨观点，通常没必要把简单交易复杂化。
