# Local Volatility Model 是什么？

Local Volatility Model 假设波动率随标的价格和时间变化。它试图拟合市场上的波动率曲面。

用途：

- 复杂期权定价；
- 风险管理；
- 与 Black-Scholes 常数波动率假设对比；
- 分析不同执行价和期限的 IV。

新手不用掌握模型推导，但要知道市场并不存在一个统一 IV。不同执行价、不同到期日的期权，隐含了不同波动率。
