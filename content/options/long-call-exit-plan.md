# 买入 Call 退出计划怎么写？

买入 Call 之前，最好先决定怎么退出。否则价格一波动，很容易从交易计划变成情绪赌博。

退出计划至少包括：

1. 盈利到多少先减仓或平仓。
2. 亏损到多少必须离场。
3. 剩余到期时间低于多少不再持有。
4. IV 下跌后是否提前退出。
5. 是平仓，还是滚到更远到期日。

买入 Call 不一定要等到到期，更不需要等到最大收益。

期权时间价值会持续衰减，判断错方向、错时间、错波动率，都可能亏钱。
