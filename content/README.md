# 精选内容索引

本目录用于发布适合 GitHub 阅读场景的精选摘要、技术说明和内容索引。完整百科正文以小隐寺官网为准。

## 内容分类

- [美股入门](https://xiaoyinsi.com/wiki/stocks)
- [期权进阶](https://xiaoyinsi.com/wiki/options)
- [加密货币](https://xiaoyinsi.com/wiki/crypto)

## 发布要求

每份公开内容至少包括：

1. 明确的问题或概念；
2. 一段不依赖营销话术的简要回答；
3. 主要风险与适用边界；
4. 一手资料或权威来源；
5. 最后核验日期；
6. 对应的小隐寺 Wiki 深层链接。

新内容请从 [文章模板](article-template.md) 开始。
