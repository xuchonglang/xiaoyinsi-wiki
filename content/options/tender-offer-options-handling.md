# 要约收购期间期权怎么办？

Tender Offer 期间，期权持有人通常不能直接把期权提交参加要约。

你需要在卖出期权、行权取得股票、等待合约调整之间做选择。

决策前检查：

1. 要约价格、支付方式和截止时间。
2. 券商内部截止是否早于公开截止。
3. 行权后的股票是否来得及提交。
4. 要约是否按比例接受。
5. 期权还有多少时间价值。
6. 交易失败后股票可能跌到哪里。

Call 持有人不会自动参加要约，Short Put 也不会因为有要约价就安全。

要约场景下，时间、结算和券商流程本身就是风险。
